Funcionamiento interno del computador

Hoy en día las computadoras contienen millones de componentes electrónicos básicos. La estructura así como el funcionamiento de una computadora son en esencia sencillos. Existen cuatro funciones básicas que un computador realiza:

  • 1) Procesamiento de datos.
  • 2) Almacenamiento de datos.
  • 3) Transferencia de datos.
  • 4) Control. Instrucciones

    • Una computadora debe ser capaz de procesar, almacenar y transferir datos así como debe de existir un control de estas tres funciones, esta función es ejercida por el usuario que proporciona las instrucciones al computador. Se denomina instrucción al conjunto de datos insertados en una secuencia estructurada o específica que el procesador interpreta y ejecuta.
