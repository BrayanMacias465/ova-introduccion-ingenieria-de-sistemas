Software de aplicacion

El software de aplicación ayuda al usuario a completar una tareas especifica, ya sea que la tarea implique generar texto o gráficos, trabajar con números, realizar gráficos o preparar una presentación.

Estos programas son empleados por el usuario para simplificar su vida, ya sea si decide escribir un libro. Una tesis o un memorándum, o para poder manejar grandes volúmenes de datos; o aquellos programas que le sirvan para emitir una nomina de una empresa. Estos programas son los mas solicitados por la mayoría de usuarios de computadoras debido a los beneficios o servicios que reciben de ellos.

Existen también programas de aplicación que brindan al usuario otro tipo de servicios; por ejemplo, los juegos por computadora son muy estimados, ya que gracias a ellos el usuario obtiene diversión y esparcimiento.

TIPOS DE SOFTWARE DE APLICACION

  • Procesador de texto
  • Procesador hoja de calculo
  • Administrador de base de datos
  • Software de presentacion
